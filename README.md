# TanksGame

8bit Tank style game. 

C++, CMake, SFML.

Controlls:

<--------------------------------------->

  FirstPlayer(Green tank):
  
    Move up - W
    
    Move right - D
    
    Move down - S
    
    Move left - A
    
    Shoot - Space
    
    
  SecondPlayer(Blue tank):
  
     Move up - Arrow up
     
     Move right - Arrow right
     
     Move down - Arrow down
     
     Move left - Arrow left
     
     Shoot - Num0
     
   General:
   
     Restart game (regenerate map) - R
     
     Exit - Esc
   
<--------------------------------------->
